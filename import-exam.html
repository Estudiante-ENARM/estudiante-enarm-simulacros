Importar examen ENARM

Usa este panel solo tú (admin) para convertir un JSON en: sección + examen + casos clínicos + preguntas dentro de Firestore.

Solo administrador
Inicia sesión con tu correo de administrador. Las escrituras en sections, exams y questions solo se permiten a usuarios con role = "admin" en la colección users.
• Si incluyes "sectionId", se usará esa sección existente.
• Si solo incluyes "sectionName", se buscará por nombre y si no existe se creará.
• Cada elemento de cases[] creará un documento en questions.
